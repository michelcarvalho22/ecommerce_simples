## Instalação

### Clone o projeto
```
https://github.com/RamSheepTech/usepointmix.git
cd usepointmix
```

### virtualenv
```
virtualenv env -p python3
```
Linux
```
source env/bin/activate
```
Windows
```
env\Scripts\activate.bat
```

## Instale as dependencias
```
pip install -r requirements.txt
python manage.py runserver
